### A Simple Todo App

This app follow a pattern which is [Redux-Ducks Pattern](https://github.com/erikras/ducks-modular-redux).

To install this app:
```
git clone https://github.com/rasitiri/react-redux-todo-app.git
```
```
cd react-redux-todo-app
```
```
npm install
```
```
npm start
```
That's it!
